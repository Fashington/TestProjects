1. Метод Tothequeue добавляет плейлист в очередь на воспроизведение.
2. Метод BrowseDisk отображает медиафайлы в конкретной директории;
    CreateNewPlayList - создает новый плейлист;
    PlayFile - воспроизводит медиафайл;
    PlayQueue - воспроизводит очередь медиафайлов;
    StartPlayList - воспроизводит плейлист;
    SortStash - сортировка медиафайлов, полученных после использования метода BrowseDisk. 
        Сортировка возможна как по имени, так и по типу файлов;
    AddToQueue - данный метод вызывает метод AddTQ класса PlayQ, который,добавляет файл в 
        очередь на воспроизведение;
    DeleteFromQueue - данный метод вызывает метод DeleteFQ класса PlayQ - удаление файла из очереди 
        на воспроизведение.
3. QueuePlay - воспроизведение очереди медиафайлов.
