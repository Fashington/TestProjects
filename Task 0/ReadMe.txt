1. Метод Ttq добавляет плейлист в очередь на воспроизведение.
2. Метод BrowseDisk отображает медиафайлы в конкретной директории;
    CreateNewPL - создает новый плейлист;
    PlayF - воспроизводит медиафайл;
    PlayQ - воспроизводит очередь медиафайлов;
    PlayL - воспроизводит плейлист;
    SortStash - сортировка медиафайлов, полученных после использования метода BrowseDisk. 
        Сортировка возможна как по имени, так и по типу файлов;
    Atq - данный метод вызывает метод AddTQ класса PlayQ, который,добавляет файл в 
        очередь на воспроизведение;
    Dfq - данный метод вызывает метод DeleteFQ класса PlayQ - удаление файла из очереди 
        на воспроизведение.
3. QPlay - воспроизведение очереди медиафайлов.
